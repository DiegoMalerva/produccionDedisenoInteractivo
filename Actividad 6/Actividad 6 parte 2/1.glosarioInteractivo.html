Dominio:
Es el nombre único y legible por humanos que se utiliza para identificar un sitio web.
Diseño Interactivo:
Proceso que busca optimizar la comunicación entre el usuario y las interfaces digitales.
URL:
La dirección única que se utiliza para acceder a una página web en la web.
Hospedaje web:
El servicio que proporciona espacio en un servidor para alojar un sitio web y hacerlo accesible en internet.
Cookies:
Pequeños archivos de texto almacenados en el navegador del usuario que se utilizan para rastrear información y personalizar la experiencia en un sitio web.
